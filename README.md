# Landing Page, Dra. Daniele Murga

Demo de landing page para a Dra. Daniele Murga, dermatologista, nutróloga e médica integrativa em Campo Grande, RJ.

## Sobre o projeto

Página estática em HTML e CSS puro, sem framework, pensada como demonstração visual pra substituir o site atual, feito numa ferramenta pronta com limitações técnicas de SEO e estrutura.

## Estrutura

```
index.html      Estrutura e conteúdo da página
style.css       Paleta, tipografia e layout
assets/         Fotos e logo usadas no site
```

## Conteúdo

- Início, com apresentação e chamada pra agendar consulta
- Sobre mim, formação e credenciais da médica
- Especialidades, Dermatologia Estética e Medicina Integrativa
- Detalhamento de procedimentos por especialidade
- A clínica, com carrossel de fotos do espaço, agrupadas por ambiente
- Depoimentos de pacientes
- Contato, com endereço, telefone e link direto pro WhatsApp

## Tecnologia

HTML5, CSS3 e Javascript puro, sem dependência externa além da fonte do Google Fonts. Responsivo, com layout ajustado pra mobile, tablet e desktop.

## Como rodar local

Abre o `index.html` direto no navegador, ou usa um servidor local simples:

```
python3 -m http.server 8080
```

Depois acessa `http://localhost:8080` no navegador.

Publicado via GitHub Pages, direto da branch `main`.
